# Doe Rey Me
simple script to learn the equivalence between english and french music notation



## Usage

```bash
./doe-rey-me.py
```

The script give you an english note (c, d, e, f, g, a or b) and you have to enter the corresponding french note (do, r√©, mi fa, sol, la or si)
